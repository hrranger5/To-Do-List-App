

# 📝 To-Do List App 

A professional, user-friendly To-Do List app built using **HTML**, **CSS**, **JavaScript**, and **Bootstrap 5**.

## 🚀 Features
- ✅ Add new tasks
- 🖊️ Edit tasks
- ☑️ Mark as completed
- ❌ Delete tasks
- 💾 Saves to browser localStorage
- 🎨 Responsive, modern UI using Bootstrap 5
- 🧠 Font Awesome icons for better UX
